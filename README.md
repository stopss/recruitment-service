# 원티드 백엔드 코스 4차 사전 과제
원티드 프리온보딩 사전 과제입니다.
```
프로젝트 실행
$ npm run start:dev
```  
</br>

## 개발 환경
* Language: Typescript
* Platform: Node.js
* Framework: NestJS
* ORM: TypeORM
* DBMS: MySQL

## Git Convention
* feat: 새로운 기능 추가
* fix: 이미 있는 코드를 수정하거나 버그를 수정한 경우
* refacotr: 코드 리팩토링
* test: 테스트 코드
* chore: 개발에 영향을 가지 않는 부분을 수정한 경우  
</br>
